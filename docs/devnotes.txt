

Si la instalación es correcta se pueden listar los idiomas soportados de la siguiente manera:


```
$tesseract --list-langs
List of available languages (3):
eng
esp
osd
snum
```


Con el servidor encendido, en otra terminal se puede hacer una petición de servicio de OCR para convertir una imágen  en texto.

```
$curl -X POST http://localhost:5000/gems/ocr -d  '{"image": "data/test.jpg", "lang":"eng"}' -H "Content-Type: application/json"

{"text":"This is a lot of 12 point text to test the\nocr code and see if it works on all types\nof file format.\n\nThe quick brown dog jumped over the\nlazy fox. The quick brown dog jumped\nover the lazy fox. The quick brown dog\njumped over the lazy fox. The quick\nbrown dog jumped over the lazy fox."}

```
